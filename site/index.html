An eBay SDK for PHP

Features

  • Compatible with PHP 5.5.0 or greater.
  • 64 bit version of PHP recommended as there are some issues when using the SDK with 32 bit.
  • Easy to install with Composer.
  • Compliant with PSR-1, PSR-2 and PSR-4.
