Item Specifics are name value pairs that describe aspects of an item. For example, A T-shirt could have an Item Specific that describes its size. In this instance the name is Size and the values could be SmallMedium or Large Note that you can only assign one value per name unless GetCategorySpecifics or GetItemRecommendations indicates that it supports multiple values.
