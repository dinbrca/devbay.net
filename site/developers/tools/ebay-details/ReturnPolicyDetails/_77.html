Germany (77)

Features available to the seller when specifying the return policy for an item.

Feature Available
Sellers can enter a text description. Yes
Sellers can specify a European Article Number (EAN). No

Sellers can specify whether they accept returns or not.

Description ReturnsAcceptedOption
Verbraucher haben das Recht, den Artikel unter den angegebenen Bedingungen zurückzugeben. ReturnsAccepted
Der Artikel kann nicht zurückgegeben werden ReturnsNotAccepted
